A look at Algorithms

The quickest and most efficient way in sorting a set of data is using QuickSort algorithm because it chooses an element from the array and compares every other element in the array pivot by placing all elements greater than the pivot into one subgroup and all elements that are smaller into another subgroup. Each subgroup is divided until it only contains one item and then all items are sorted in order.
